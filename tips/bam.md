XA:Z:A1.1,-918,150M,0;

| 字段部分     | 示例值 | 含义                          |
| ------------ | ------ | ----------------------------- |
| 字段名与类型 | XA:Z:  | 记录次要比对的字符串字段      |
| 参考序列     | A1.1   | 次要比对所在的染色体/序列名   |
| 位置与方向   | -918   | 从G1.1的第918碱基开始反向比对 |
| 比对模式     | 150M   | 读段150个碱基全部参与比对     |
| 错配数       | 0      | 该次要比对位置零错配          |
| 分隔符       | ;      | 表示一个次要比对的结束        |



## 字段概览

```bash
89M61S
NM:i:0  MD:Z:89 MC:Z:150M       AS:i:89 XS:i:0  RG:Z:project   SA:Z:insert,4027,-,77S73M,35,0;
```



| 字段                            | 类型   | 值           | 含义             |
| :------------------------------ | :----- | :----------- | :--------------- |
| NM:i:0                          | 整数   | 0            | 编辑距离         |
| MD:Z:89                         | 字符串 | 89           | 错配/删除字符串  |
| MC:Z:150M                       | 字符串 | 150M         | 配对 读段的CIGAR |
| AS:i:89                         | 整数   | 89           | 比对得分         |
| XS:i:0                          | 整数   | 0            | 次佳比对得分     |
| RG:Z:project                    | 字符串 | project      | 读组标识         |
| SA:Z:insert,4027,-,77S73M,35,0; | 字符串 | 补充比对信息 |                  |



```
insert,4027,-,77S73M,35,0;
```

- **`insert`**：参考序列名称（可能是某个插入序列或特殊区域）
- **`4027`**：比对起始位置（4027号碱基）
- **`-`**：反向比对
- **`77S73M`**：CIGAR字符串
  - `77S`：77个碱基是**软剪接**（soft-clipped），意味着这77bp没有比对到参考序列上
  - `73M`：接下来的73个碱基完全匹配
- **`35`**：该补充比对部分的映射质量（MAPQ）
- **`0`**：该部分的编辑距离（NM值），0表示这73个匹配的碱基完全匹配
